## Запуск проекта

Клонирование проекта

```bash
  git clone https://github.com/Barry-Kenan/personal-diary.git
```

Переходим в папку с проектом

```bash
  cd personal-diary
```

Устанавливаем зависимости

```bash
  npm install
```

Запускаем проект

```bash
  npm start
```
